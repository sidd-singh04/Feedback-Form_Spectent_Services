#  Feedback Form System – Build, Break & Explain

## Description
A full-stack web application that allows users to submit feedback through a simple form. Built using React.js for frontend and Node.js with Express for backend. The system focuses on form handling, validation, API communication, and basic system testing.

---

## Features
- Submit Name, Email, and Feedback
- Form validation (required fields + basic email check)
- Backend API for handling submissions
- Success and error message display
- Loading state during submission

---

## Tech Stack
Frontend: React.js, CSS  
Backend: Node.js, Express.js  
Tools: VS Code, Git & GitHub
